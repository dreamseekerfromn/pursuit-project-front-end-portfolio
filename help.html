HELP PAGE

  • In the "Number of Trial" input field, it can accept only integer, no float and no negative number or zero.
  • For the Quest Rank (or you can say, difficulty), if the monster does not appear in that rank quest, the page will show an error message.
  • Clicking a list will show the detail page.
  • Hovering the monster icon in the detail page will show a tooltip for the weaknesses
  • The bottom box will display a random monster's information
  • The API does not have all of the info, so be aware about that.
